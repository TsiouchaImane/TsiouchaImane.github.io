Agile Scrum Master

Scrum is a light method, easy to understand but difficult to master. It is based on three pillars:
- Transparency, in fact, the important elements of the project will be visible to all the individuals of each team. This pillar implies that the language used is understandable by all.
- Inspection, a step intended to monitor the progress of the project against a sprint goal. A sprint is a period during which the people in charge of the project must design a specific element of the deliverable before revision. A sprint lasts an average of two weeks and can last up to a month.
- And finally, adaptation, representing an adjustment to the deliverable if it has drifted outside acceptable limits.

  • Network security

    The study and Configuration of Access control List in a real Network

    Network security

    The study and Configuration of Access control List in a real Network

    This Project describes how IP access control lists (ACLs) can filter network trafic. It also contains brief descriptions of the types of IP access control lists, availability of features, and an example of use on a network. ACLs can also be used for purposes other than filtering IP trafic, for example, to define NAT or encryption trafic, or to filter non-IP protocols, for example AppleTalk or IPX.
    To filter network trafic, ACLs check whether packets are routed or blocked at the router interface. The router examines each packet to determine whether to ship or release the packet based on the criteria specified in the ACL.

    The criteria in the ACL list include:
    - Trafic source address
    -Trafic destination address
    -Upper layer protocol
    In order to build an ACL we :
    -Create an ACL list.
    - Apply the ACL to an interface.

    The IP ACL is a sequential collection of authorization and denies the conditions that apply to an IP packet. The router tests the packets based on the conditions present in the ACL, one after the other. The IP ACLs that can be configured in Cisco IOS are:
    -Standard ACLs
    -Extended ACLs
    -ACLs named IP
